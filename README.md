# AI-ML-internship-tasks
About:
This repository contains AI/ML internship tasks completed using Python, 
Jupyter Notebook, Pandas, Seaborn, and Scikit-learn.

## Task 1: Exploring and Visualizing Iris Dataset
- Objective: Load, inspect and visualize Iris dataset
- Dataset: Iris Dataset (Seaborn built-in)
- Libraries: Pandas, Seaborn, Matplotlib
- Key Steps: Data loading, EDA, Scatter plot, Histogram, Box plot


## Task 3: Heart Disease Prediction
- Objective: Predict whether a person is at risk of heart disease
- Dataset: Heart Disease UCI Dataset (Kaggle)
- Models: Logistic Regression, Decision Tree
- Results:
  - Logistic Regression Accuracy: 78.8%
  - Decision Tree Accuracy: 80.4%
- Key Findings: Decision Tree worked better with categorical data


## Task 6: House Price Prediction
- Objective: Predict house prices using property features
- Dataset: House Price Prediction Dataset (Kaggle)
- Models: Linear Regression, Gradient Boosting
- Results:
  - MAE: ~$244,000
  - RMSE: ~$280,000
- Key Findings: The dataset was made up at random, so the MAE is high.
